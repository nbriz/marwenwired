Marwen WIRED!
@RT 4 the INTERNET



“The single most important thing you need in order to have a career in the arts is persistence. The second most important thing you need is talent. The third most important thing is a grounding in how the online world works. Its that Important” — Cory Doctorow



Many of us live online, we use the Internet everyday for shopping, social media, gaming, news and so much more. The same technologies used to make all of those platforms can also be used to make and exhibit art. The browser is a blank canvas, with the right tools and skills you can turn the browser into an interactive and generative art work accessible to nearly anyone around the world instantly! In this course we’re learn how the Internet works and how to code everything from a simple website to dynamic Internet Art using HTML, CSS and JavaScript.

Lessons




What Is the Internet really


what is the Internet? how does it work? does anyone own it? who makes the rules? who’s in control? why do i have to pay for it? who’s making money? the answers to these questions will set the stage for the conversations we will be having in class.

Day 01 Notes

Hypermedia on the WWW


from Vannevar Bush to Ted Nelson to Tim Berners-Lee to u. we’ll discuss what the world wide web is and how it came to be as well as discuss the concepts of “hyper” media broadly. we’ll learn the fundamentals HTML ( hypertext markup language ) and how to use code editors and browsers to make web pages.

Day 02 Notes

demystifying the "Cloud"


at the 2011 Apple Worldwide Developer’s Conference Steve Jobs announced, “we’re going to move the digital hub, the center of your digital life, into the cloud”, but what xactly is “the cloud” and are we sure we want to move our “digital life” there? we’re going to be learning FTP ( file transfer protocol ) and how we can take control of over our own “clouds.”

Day 03 Notes

Programming the Web


we’ll learn the basix of programming ( using JavaScript, the programming language of the web ) such as data, variables, functions. we’ll look at how different artists ( even b4 code ) use instructions to produce very different kinds of work.

Day 04 Notes

Interaction + Animation


we’ll learn the programming concepts of “loops” and “conditional statements” and how these can bring elements of interactivity and generative animations to our drawings. we’ll look at artists who use the kind of interactivity possible on the web to make participatory art

Day 05 Notes

Paint by Numbers


we’ll continue to build on our programming skills by learning even more ways to draw arcs, lines and curves. we’ll also learn how to make our code more efficient by leveraging the computer’s amazing ability to do things really fast using “while loops” and “for loops”.

Day 06 Notes

Algorithms run our world


we’ll expand on what we’ve learned about “loops” and “conditional statements” and how in combination with mathematical functions these can be used to write “algorithms”, we’ll talk about the role algorithms play in today's world and how in our case we can use them to create generative and interactive drawings.

Day 07 Notes

Coding Nature


programmers have been writing algorithms that imitate nature for years. creating code that models the way gravity works and the ways birds flock and much more. today we’ll be learning how to integrate some of the more popular of these algorithms into our code.

Day 08 Notes

open studio


we’ll look at some more web art and specifically how these artists install their work in more traditional galleries. we’ll take sometime to polish our final projects and discuss how we’ll be presenting them in the final exhibition.

Day 09 Notes

epilogue


we’ll be using the last day of class to finish up our projects and tie up any loose ends. we’ll also be talking about where we can go from here. how can we continue our internet art practice after marwen. where can we host our site? where can we go to learn more? where can we go to get help?

Day 10 Notes

bonus lessons




CSS: form follows function;


BONUS: CSS ( Cascading Style Sheets ) which is the language used to ad form to the functional elements of a markup language ( ie. HTML ). CSS lets u specify rules that determine how the content of a page will appear, for ex: the color, size and typeface of your text.

Bonus Notes

X, Y, Z (the third dimension)


the digital world can have an multiple dimensions, until now we’ve only been working in two. using the popular javascript library three.js we can add a third dimension to our projects and learn about all that comes with that ( cameras, meshes and more )
