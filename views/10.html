where to go from here?



there's libraries, templates and code-snippets for everything out there ...

...But you need to know how to find them [and] what they are called. Code isn't just obscure commands in a file. it requires you to have a map in your head, to know where the good libraries, the best documentation, and the most helpful message boards are located. If you don't know where those things are, you will spend all of your time searching, instead of building cool new things" — Paul Ford



there’s so much more to learn! some of you might be interested in cleaning up your home pages. there’s a great tutorial series called “Don't Fear the Internet” which goes through HTML && CSS web design in a few short videos. I strongly suggest watching all of those if you want to get your web skills strong. if your the kind of person that prefers learning from books then i strongly recommend the book HTML & CSS by John Duckett

if you want to dive deeper in to JavaScript, there’s SO MUCH more to learn :D, again if you like books John Duckett’s got one for JavaScript too called JavaScript & JQuery. another great JavaScript book ( for those of you who want to master the language ) is Eloquent JavaScript ( you can buy it in paper but there’s an online version of the whole book here ).

while your working on your website and/or net art there’s a few websites that have great references. w3schools has all the info you could ever want on every HTML tag. CSS-Tricks has a great reference for all the CSS properties ( as well as lots of cool css tricks as the name implies ) and Mozilla ( the foundation that makes the firefox browser ) has some beautifully organized resources on JavaScript. Also if you ever get stuck trying to figure something out you should check out the site StackOverflow. it’s a forum where folks ask coding questions and other people answer them. there’s a good chance someone has asked your question before ( and if not, you can ask it yourself :) >> people can be super nice and also super mean on forums, it’s important to follow the rules of any community before jumping in. on StackOverflow, when asking a question here’s a couple rules to keep in mind ( and if you do, lots of friendly people will quickly answer your question )

  1. make sure no one else has already asked the question ( search the site and see )
  2. make sure the question you are asking is clear and includes the code you’re having issue with
  3. don’t copy all of your code! just the part you’re having trouble with
  4. if someone answers the question right, mark it as “correct” ( they appreciate it cause they get points, but also it lets other people know which is the right answer )


when all else fails, i’m only an email away ( nickbriz@gmail.com ) don’t hesitate to send me an email if you have any questions about the Internet, or if you want to send me any cool net art your making :)





renewing your website ( 1 year from now )



BE WARNED! your domain name is only registered to you for 1 year! you will have to pay to renew it ( somewhere around $10-$12 ) yourself every year. similarly, our server will only be up and running for 1 year! so you will have to transfer your files to a new server if you want to keep your website up for after a year. the steps below will walk you through the process of renewing + transferring



step 1 your current domain is registered through a company called namecheap, you can visit their website and sign into your account. your username is your first name + wired ( so for example zac’s user name is zacwired, mecca’s user name is meccawired … there’s one exception, evanwired was already taken so evan’s user name is evanwired1 ) then use your ftp password ( the one you made in class ) to log in.




step 2 the marwen server will go down in a year, but you can move your website to one of namecheap’s servers ( the company you pay for server space is usually called your “host” ). once you are logged in click on “Shared Hosting” in the “Hosting” menu.




step 3 there are lots of different payment options. you’ll need a credit card ( or other digital payment method ) to purchase a year of hosting, so if you don’t have your own you’ll likely need your parent/guardian’s help for this part. choose a plan that seems right for you.




step 4 once you pick a plan you’ll see this window. choose the “use a domain i own with namecheap” option. on the next page you should see your domain name. click the SELECT button beside your domain.




step 5 when you finish and pay, you should now be able to log onto your website using cyberduck as always ( the only difference is that your website will now be “hosted” on namecheap’s servers instead of the marwen server ). if you have any issues logging in you can chat with a customer service rep on namecheap’s website ( they’re very helpful, i’ve chatted with them before )
